# 什么是webpack
本质上，webpack是一个用于现代JavaScript应用程序的 **静态模块打包工具**，当webpack处理程序时，它会在内部从一个或多个入口点构建一个[依赖图](https://webpack.docschina.org/concepts/dependency-graph/)，然后将你项目中所需的每一个模块组合成一个或多个bundles,它们均是静态资源，用于展示你的内容
